# Gold Line Services Next.js Website v2

## Що вже додано
- Сучасний Next.js сайт
- Логотип
- Фото з Unsplash як тимчасові
- Services, About, Gallery, Process, Contact
- SEO title/description
- Форма заявки через Formspree placeholder

## Запуск
npm install
npm run dev

## Для робочої форми
1. Зайти на formspree.io
2. Створити форму
3. Замінити YOUR_FORM_ID у app/page.tsx
